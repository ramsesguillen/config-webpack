# Cconfiguración de webpack

Este es el proyecto inicial para crear aplicaciones utilizando Webpack.

## Notas:
Recuerden reconstruir los módulos de Node

```
npm install
```

Y para construir el build(mode: productotion), recuerden:
```
npm run build
```
(mode: development)
```
npm rub build:dev
```

Correr el servidor.
```
npm start
```
